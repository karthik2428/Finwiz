# Finwiz 💰

Finwiz is a finance management application designed to help users track expenses, manage budgets, and gain insights into their financial habits.

## 🚀 Features

* Expense tracking
* Budget management
* Financial insights & analytics
* User-friendly interface

## 🛠️ Tech Stack

* Frontend: React
* Backend: Node.js, Express
* Database: MongoDB

## 📦 Installation

```bash
git clone https://github.com/karthik2428/finwiz.git
cd finwiz
npm install
npm start
```

## 📌 Usage

* Add your expenses
* Set budgets
* Track your financial growth
